Expandable Textbox

Add .expandableMinimal to make the textbox expandable and of initial minimal size. Or define the width yourself:

.dwidget .dijitTextBox.expandableMinimal{
  width:0px;
}

.dwidget .dijitTextBoxFocused.expandableMinimal{
  width:200px;
}

Add .queryTextBox to put a query icon in the textbox. Add .roundTextBox to make it rounder.

Filtering Selects Cool and easy to use

The behaviours from the TextBox are available in the filtering select since it inherits from the TextBox

Filtering Select

With a preset value

Expandable

The filtering select is created programmatically like: 

new FilteringSelect({
  store:store,
  'class':'expandable queryTextBox'
}, 'filtering_select_expandable');
